## Set-Up Instructions

##### 1. Scrape data from the [offical Tour website](https://www.letour.fr).

```bash
$ python scrape_tdf_data.py
```

##### 2. Analysis

```bash
$ jupyter notebook tdf_analysis.ipynb
```
